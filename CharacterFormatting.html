• The < b> tag used to make the text bold or increase the weight of the text
  • The < strong> < /strong> is same as < b> tag.
  • The < i> < /i> is used to make text italic or inclined toward the right
  • The < em> < /em> tag is same as < i> tag
  • The < u> < /u> tag is used underline the text
  • The < ins> < /ins> tag is same as < u> tag.
  • The < del> < /del> tag used to put horizontal line through text
  • The < strike> < /strike> tag is same as < del> tag(deprecated).
  • The < big> < /big> tag is used make text bigger than rest(deprecated).
  • The < small> < /small> is used to make text smaller than the rest.
  • The < sup> < /sup> is used to make the text super script.
  • The < sub> < /sub> is used to make the text sub script.
  • The < mark> < /mark> tag is used to mark the text like highlighter.
