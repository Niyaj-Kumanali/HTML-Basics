Media tags:

HTML uses media tags to display the audio & video files on the webpages.
Attributes:
  • src: It contains the url or path
  • controls:
    • It is used to show the controls for the video & audio.
    • Without controls the audio tag will not be rendered.

  • muted: It is used to mute the audio & video by default.

  • autoplay:
    • It is used to autoplay the video and audio
    • In some browser the autoplay doesn't work for audio.

  • loop: It is used to loop the audio & video.

  • poster: It is used to add poster to video. It disapeers after video is played
  • height: It is used to set height for the video.
  • width: It is used to set width for the video.
