HTML structure:

< !DOCTYPE html>
< html>
< head>
< meta name="keyword" content="html, css, js" />
< meta name="description" content="The website give information about front end development" />
< title>Page Title< /title>
< /head>
< body>
< h1>My First Heading< /h1>
< p>My first paragraph.< /p>
< /body>
< /html>
  • The < !DOCTYPE html> declaration defines that this document is an HTML5 document
  • html tag:
    • The < html> element is the root element of an HTML page
    • very html file start and end with < html> tag
  • head tag:
    • The < head> element contains meta information about the HTML page
    • Collection of < meta> tag about the html document.
  • title tag:
    • The < title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
    • It represent the name or title of the pages on tab of the browser.
  • body tag:
    • The < body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
    • It represent content of the pages.
  • meta tag:
    • < meta> tags provide additional information about a webpage to the search engines which results in better search results.
    • With the help of < meta> tag the browser gets to know what is that perticular website all about.
    • The information about the website contain keyword, description, title, author etc.

Heading Tags :

  • HTML uses the h1, h2, h3, h4, h5, h6 in order to display heading on the webpage.
  • The difference between all these tags are the font size goes on decreasing from h1 to h6.
  • All heading tags are bold by default.

  • HTML headings are defined with the < h1> to < h6> tags.
  • < h1> defines the most important heading.
  • < h6> defines the least important heading.
  • Search engines use the headings to index the structure and content of your web pages.
  • Users often skim a page by its headings.
  • It is important to use headings to show the document structure
  • < h1> headings should be used for main headings, followed by h2 headings, then the less important < h3>, and so on.

Paragraph tag:

  • The < p> tag defines a paragraph.
  • Browsers automatically add a single blank line before and after each < p> element.
  • The < p> tag is used to display the paragraph on the webpage.

Attributes:

Attributes or properties tha are written in the starting tag which gives extra information about the tag and also has a ability to change the behavior of the tag.
