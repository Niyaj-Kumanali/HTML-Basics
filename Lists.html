Lists:

HTML uses 3 types of lists:

  1. Unordered List:
    • Use to display elements which has no perticular order.
    • All the elements will be displayed with be help of bullets
    • The style of the list can be changed using 'type' attribute like (disc(default), circle, square).
    • Syntax:
      < ul>
      < li>item 1< /li>
      < li>item 2< /li>
      < /ul>
  2. Ordered List:
    • It is used to display all the list items in a perticular order.
    • In order list all the list items will be displayed with help of the numbers like 1(default), I, i, A, a.
    • Syntax:
      < ol>
      < li>item 1< /li>
      < li>item 2< /li>
      < /ol>
  3. Description List:
    • < dl> tag is used to display list of description for terms.
    • The terms are written within < dt> tag.
    • The description for the terms are written within < dd> tag.
    • Syntax:
      < dl>
      < dt>item 1< /dt>
      < dd>item 2< /dd>
      < /dl>
