Tables:

  • HTML uses < table> tag to represent the data in the forms of rows and columns on the webpage
  • The rows for the table are created using < tr> tag.
  • The < th> is used to display the headings for the table.
  • The data for heading are written within < td> tag.
  • Syntax:
    < table>
    < tr>
    < th> < /th>
    < /tr>
    < tr>
    < td> < /td>
    < /tr>
    < /table>
  • Ex:

    • Sl. No. Name Age
    1 Niyaj Ahmad 23
    2 Nawaz Kumanali 23
    3 Pattu Pol 23

  • rowspan & colspan
    • These are attributes that are used with < td> & < th> tag, which allows a single cell to occupies more than one columnes or rows.
